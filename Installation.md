# 安装

*本部分介绍如何安装Xdebug*

如何安装 Xdebug 取决于您的系统。有以下几种可能：

* Linux使用包管理器比如apt、yum等安装；
* Linux使用PECL安装；
* macOSX使用homebrew工具,通过PECL安装；
* Windows通过安装向导帮助安装；
* <font color="green">类Unix操作系统</font>，源码安装。
